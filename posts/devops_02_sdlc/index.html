#SDLC(software development Lifecycle) ##whats design development test high quality

StageCore ActivityOutput / Purpose
PlanningGather requirements from customers; conduct feasibility analysis.Determine project scope and goals.
DefiningTranslate requirements into clear documentation.Produce specifications and requirement documents.
DesigningCreate the system architecture and module structure.High-Level Design (HLD) (System architecture, component interaction);Low-Level Design (LLD) (Implementation details, algorithms).
BuildingWrite and implement the code.Functional software modules.
TestingVerify software functionality and identify defects.Ensure the software meets the expected high quality standards.
DeploymentRelease the software to the production environment.The final product ready for user access.

DevOps (Development + Operations) is a culture, practice, and toolset aimed at shortening the system development life cycle while ensuring continuous, high-quality delivery.
