我的第一篇文章

November 11, 2025

欢迎来到我的博客！这是使用 Hugo 和 GitHub Pages 创建的第一篇文章。

关于这个博客

这个博客使用以下技术栈构建：

  • Hugo: 快速、现代的静态网站生成器
  • GitHub Pages: 免费的网站托管服务
  • GitHub Actions: 自动化部署流程
  • Ananke 主题: 简洁美观的 Hugo 主题

下一步

您可以：

  1. 编辑这篇文章，添加更多内容
  2. 创建新文章：hugo new content posts/your-post-name.md
  3. 本地预览：hugo server -D
  4. 推送到 GitHub 后，网站会自动部署

祝您写作愉快！
